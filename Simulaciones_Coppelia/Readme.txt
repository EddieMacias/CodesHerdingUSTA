Instrucciones para ejecutar los Modelos

Para modelo 01 y modelo 02

1) Abrir el Archivo Quadcopterswarm.ttt Con el software Coppelia
2) En el software Coppelia Dar Clic en Start Simulation  >
3) En coppelia debe aparecer el mensaje [/Controller@childScript:info] Handle de los targets obtenido correctamente
3) Abrir el Archivo QuadcopterMove.m con el Software Matlab
4) Una vez se haya ejecutado el Paso 2 y la simulacion se esté ejecutando en coppelia ejecute el script de Matlab (f5)
5) Una Vez Matlab detecte la conexion UDP con Coppelila debe aparecer 
	Note: always make sure you use the corresponding remoteApi library
	(i.e. 32bit Matlab will not work with 64bit remoteApi, and vice-versa)
	Conectado a CoppeliaSim
	Handle del target obtenido correctamente
6) Al finalizar la Simulacion desde Matlab se finalizara el coppelia y aparecern los resultados!!
